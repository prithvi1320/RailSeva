# **App Name**: RailSeva

## Core Features:

- Mobile OTP Authentication: Secure user authentication using mobile number and OTP verification.
- Complaint Registration: Form to submit complaints with journey details (PNR, Train No, Station Name), grievance description, voice-to-text input and file uploads.
- Voice-to-Text Assistant: AI-powered tool that transcribes user's voice input into text for the grievance description, enhancing user experience and accessibility.
- Complaint Categorization: Categorize complaints into types such as Train Related, Station Related, Medical Assistance, Security Assistance, Appreciation/Rail Anubhav, and Suggestions.
- Complaint Tracking: Track the status of complaints using a unique Complaint ID, with status updates such as Submitted, Assigned, In Progress, and Resolved.
- Admin Panel: Secure admin login to view, filter, and assign complaints to departments, update complaint status, and add internal remarks.
- Real-time Notifications: Send SMS/Email notifications on complaint submission and status changes. Real-time updates via Socket.io.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to evoke trust and stability, reflecting the reliability of the railway service.
- Background color: Light gray (#F0F4F8), providing a clean and professional backdrop.
- Accent color: Vivid orange (#E65100), used sparingly to highlight important actions and notifications.
- Body font: 'Inter', a grotesque-style sans-serif, for a modern and neutral look in body text.
- Headline font: 'Space Grotesk', a sans-serif, lending a techy and scientific feel to the headings.
- Use universally recognized icons from a set like Material Design Icons to ensure clarity and ease of understanding.
- Implement a clean, government-style layout with a left vertical navigation sidebar for easy access to different sections.
- Use subtle transitions and feedback animations to enhance user interaction and provide a smooth experience, such as loading indicators and form submission confirmations.
